## 4. تأكيد المزامنة

- **الوثائق:** `investment-assumptions-register.md` و `zone-c-zone-d-risk-notes.md` يعكسان الآن القيم والوصف الموجودين في `assumptions.ts` (cap 10%، رامب أ 4→50 و 450→2150، 340 يوم، Zone B 2.1M، Zone C 72% إشغال و 12% OPEX و CAPEX placeholder، Zone D 560k/شهر و 6.72M/سنة و 5.5 شهر تعادل).
- **الواجهة:** لم تُضف أرقام جديدة؛ تم فقط تخفيف العبارات التي توحي باليقين (expected / متوقع) إلى تقدير أو "من النموذج" حيث أوصى تقرير Validation.
- **مصدر الحقيقة:** لم يُغيّر أي رقم في assumptions.ts أو في المحرك؛ التعديلات كانت sync للـ docs ونصوص العرض فقط.

**ملاحظة:** تقارير `investment-sanity-check.md` و `financial-calibration-pass.md` تُركت كما هي لأنها توثّق الحالة السابقة والتغييرات عند المعايرة (before/after)، وليست وثائق "الحالة الحالية" القابلة للتشغيل.
